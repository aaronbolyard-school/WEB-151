Blue Wizard Digital's game Friday the 13th: Killer Puzzle is a puzzle game based on planning the best way to slay some innocent (and sometimes not-so-innocent) victims in order to please Jason's insane mother. There's obstacles, hazards, traps, moving victims, and other elements that increase difficult as the player progresses.
